Join us for real-time interaction in #k-slug on irc.freenode.net.

Getting started with IRC

“Internet Relay Chat (IRC) is a form of instant communication over the Internet. It is mainly designed for group (Many-to-many) communication in discussion forums called channels, but also allows one-to-one communication.” –wikipedia entry for irc

To use IRC, you need an IRC Client.

Windows IRC Clients:

  • mIRC
  • upp

Linux IRC Clients:

  • konversation
  • talksoup

Mac OSX Clients:

  • Colloquy

Multiplatform Clients:

  • pidgin
  • chatzilla
  • irssi
    • screen + irssi
  • xchat

Once you have a client installed, you will need to give it a server name to connect to IRC. Use irc.freenode.net. Once you are connected you need to join the #k-slug channel. This can be accomplished in most clients by typing /join #k-slug and hitting enter. For more help on IRC see this IRC help guide. Hope to see you online!
